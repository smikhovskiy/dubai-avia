# Plan: Dubai-Moscow Flight Finder

Приложение для поиска и ранжирования авиабилетов Дубай ↔ Москва (туда-обратно).

## Цель

Показывать не только цену и время, но и неочевидные характеристики рейса, чтобы пользователь мог выбрать действительно лучший вариант.

## Архитектура

```
[Frontend] → [FastAPI backend] → [Aviasales Tequila API]  (цены, рейсы)
                               → [AviationStack]           (задержки, отмены)
                               → [OpenFlights / SeatGuru]  (комфорт, тип самолёта)
```

## Стек

| Слой      | Технология              |
|-----------|-------------------------|
| Backend   | Python + FastAPI        |
| Frontend  | HTML + JS               |
| Кэш       | JSON-файл (потом Redis) |
| Deploy    | Локально (пока)         |

## Характеристики рейса

**Очевидные**
- Цена
- Дата и время вылета / прилёта
- Длительность
- Число пересадок
- Авиакомпания, номер рейса
- Включён ли багаж

**Неочевидные**
- Вероятность отмены рейса
- Вероятность задержки > 30 мин
- Уровень комфорта (тип самолёта, шаг кресел)
- Время суток (утро / день / красный глаз)
- Расчётная загруженность рейса

## Скоринг

```
score = 0.35 * цена
      + 0.20 * длительность
      + 0.20 * надёжность
      + 0.15 * комфорт
      + 0.10 * время_суток
```

Веса настраиваются через UI.

### Правило: аэропорт вылета (ОАЭ)

Приоритет аэропортов со стороны ОАЭ:

| Приоритет | Аэропорт | Код |
|-----------|----------|-----|
| 1 (лучший) | Dubai International | DXB |
| 2 | Abu Dhabi International | AUH |
| 3 | Al Maktoum (Dubai World Central) | DWC |

### Правило: аэропорт прилёта (Москва)

Приоритет московских аэропортов:

| Приоритет | Аэропорт | Код |
|-----------|----------|-----|
| 1 (лучший) | Шереметьево | SVO |
| 2 | Внуково | VKO |
| 3 | Домодедово | DME |

### Правило: время суток в будние дни

Если вылет в **пн–пт**, предпочтение отдаётся рейсам после **18:00** (вечерний слот).
Рейсы до 18:00 в будни получают штраф к компоненте `время_суток`.
